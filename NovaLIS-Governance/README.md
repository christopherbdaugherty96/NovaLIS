## Current Governance Posture

Nova currently operates as:
- a governed, invocation-bound system
- with active governed execution
- with explicit trust and settings surfaces
- with governed memory and continuity
- with advisory-only external reasoning

Nova still does not allow:
- autonomous execution
- background delegated triggers
- wake word runtime as live truth
- Phase 8 external execution
