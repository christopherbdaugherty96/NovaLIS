# NovaLIS Quickstart

Get Nova running locally and reach a first success quickly.

## Fast Path
1. Install dependencies.
2. Start Nova.
3. Open the local dashboard.
4. Try the first commands below.

## First Commands
1. What works today?
2. Explain what you can do
3. Summarize today's news
4. Draft an email to test@example.com

## Install
```bash
git clone https://github.com/christopherbdaugherty96/NovaLIS.git
cd NovaLIS
pip install -e .
```

## Start
```bash
nova-start
```
