# NOVA

## Nova is a Governor Over Contained Intelligence.

Nova is not an assistant.
Nova is not autonomous.
Nova is not a personality engine.

Nova is a governed intelligence appliance.

---

## Core Identity

Nova is a **Butler-Agent Under Law**.

It provides intelligence.
It does not possess authority.

All authority flows through a single Governor spine.

Intelligence may expand.
Authority does not.

---

## Architectural Principle

User → Governor → Execution
User → Contained Intelligence → Advisory Output

The Intelligence–Authority Split is absolute.

---

## What Nova Does

- Governed web search
- Structured reporting
- Explicit system control (volume, brightness, media)
- Invocation-bound speech
- Read-only environmental awareness (OS diagnostics)
- Deterministic clarification

---

## What Nova Does NOT Do

- No background cognition
- No proactive behavior
- No autonomy
- No multi-step orchestration
- No hidden execution
- No learning or adaptation
- No execution outside the Governor

---

## The Governor

The Governor is the single authority choke point.

Every action:
- Logged
- Audited
- Fail-closed
- Deterministic

If it does not pass through the Governor, it does not execute.

---

## Deep Thought

Deep Thought is contained external intelligence.

It:
- Is stateless
- Is user-invoked only
- Cannot execute
- Cannot escalate authority
- Cannot access tools directly
- Cannot persist

It provides analysis only.

Current status: Placeholder implementation using local model. Full DeepSeek integration is design-only.

---

## Design Philosophy

Nova is built on:

- Predictability
- Transparency
- Mechanical enforcement
- Sovereignty
- Calm UX
- Bounded intelligence

---

## Strategic Position

Nova is not competing with assistants.

Nova is a governance layer that contains intelligence.

It scales intelligence safely.

Authority remains singular.
