# NOVA

Nova is a governed intelligence platform.

## What Nova Is (For Users)

Nova is a personal assistant on your computer that helps you get clear answers and stay organized.

To you, it is:

- A place to ask questions and get straight answers
- A daily briefing tool for weather, headlines, and important updates
- A news helper that can read multiple sources and give one clear summary
- A follow-up helper where you can say things like "shorter," "compare those," or "explain that"
- A guide that asks for confirmation before risky actions
- A system that stays under your control and only acts when you ask

In simple terms: Nova is your calm, trustworthy "information and action desk" for day-to-day use.
