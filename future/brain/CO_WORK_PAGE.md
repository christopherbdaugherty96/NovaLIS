# Co-Work Page (Planning Only)

Defines Nova's future primary working interface.

This page represents the user's active working environment with Nova.

---

## Purpose

Provide a single place where the user can:

- see all active runs
- switch focus between runs
- view run progress
- steer runs
- review results
- continue conversation

---

## Core Sections

### Active Runs Panel

Shows:

- run title
- status
- current step
- pending approvals

---

### Focused Run View

Shows:

- Run Script
- progress
- current step
- next step
- approvals
- results

---

### Run Chat Panel

Scoped chat for the focused run.

---

### Global Chat Panel

Conversation layer for general requests.

---

## Behavior

- Switching runs updates focus without executing anything
- Runs do not auto-resume on focus
- All execution remains governed

---

## Rule

> The Co-Work page shows work. It does not grant authority.

---

## Status

Planning only. Not implemented.
