# Response System (Planning Only)

Defines Nova's structured response behavior.

## Core Flow
Clarify → Understand → Plan → Improve → Execute (if approved)

## Rules
- No direct execution from responses
- Execution must occur within a Run

## Clarification
Only when ambiguity exists.

## Understanding
Nova must restate user intent before planning.

## Planning
Nova produces a structured plan (Run Script).

## Improving
Nova suggests improvements, risks, and optimizations.

## Execution
Only after:
- plan exists
- approval is granted
- Governor allows

## Status
Planning only. Not implemented.
