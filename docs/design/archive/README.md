# Design Archive

This folder is for superseded or historical design artifacts.

These files are:
- historical
- superseded
- not current roadmap authority

Use instead:
- `docs/design/README.md`
- `docs/design/DESIGN_AUTHORITY.md`
- the relevant active phase folder under `docs/design/Phase */`

Rules:
1. Keep canonical active docs in phase folders.
2. Move superseded docs here when deletion is not possible.
3. Add a deprecation banner in old locations that points to the canonical file.
4. If an archive file conflicts with runtime truth or active phase maps, the active docs win.
