### **Week 3: Reversibility Framework**

**Deliverable 3: Reversible Action Protocol**
```python
class ReversibleAction:
    """Wrapper for any tool that modifies state"""
    
    def __init__(self, tool_fn, reversal_fn):
        self.tool_fn = tool_fn
        self.reversal_fn = reversal_fn
        self.snapshot_manager = SnapshotManager()
    
    def execute(self, params: dict) -> ReversibleResult:
        # 1. Take pre-execution snapshot
        snapshot_id = self.snapshot_manager.capture_state(params)
        
        try:
            # 2. Execute original tool
            result = self.tool_fn(params)
            
            # 3. Generate reversal script
            reversal_script = self.reversal_fn(
                original_params=params,
                result=result,
                snapshot_id=snapshot_id
            )
            
            # 4. Validate reversal actually works (test in sandbox)
            if not self.test_reversal(reversal_script, snapshot_id):
                raise IrreversibleError("Reversal validation failed")
            
            return ReversibleResult(
                result=result,
                reversal_script=reversal_script,
                snapshot_id=snapshot_id,
                validation_hash=sha256(reversal_script)
            )
            
        except Exception as e:
            # 5. Auto-rollback on failure
            self.snapshot_manager.restore(snapshot_id)
            raise
```

### **Week 4: Decision Delegation Proof**

**Deliverable 4: "Decide How, Not Whether" Implementation**
```python
class DecisionDelegator:
    """Allows LLM to choose implementation details, not core decisions"""
    
    DELEGATABLE_ASPECTS = {
        "append_to_file": [
            "formatting",      # LLM can decide: bullet points, numbering
            "timestamp_style", # ISO vs human-readable
            "metadata",        # Add tags, categories
        ],
        "fetch_data": [
            "sort_order",
            "filter_criteria",
            "presentation_format"
        ]
    }
    
    NON_DELEGATABLE = {
        "append_to_file": [
            "target_file",     # User must specify
            "core_content",    # User's exact words (can be formatted but not changed)
            "whether_to_write" # Always yes if user said "append"
        ]
    }
    
    def validate_delegation(self, user_intent: str, llm_decision: dict) -> bool:
        """Verify LLM only modified allowed aspects"""
        
        # Extract user's exact verb
        user_verb = extract_verb(user_intent)  # "append", "add", "write"
        
        # Map to tool
        tool = verb_to_tool_mapping(user_verb)
        
        # Check LLM didn't change non-delegatable aspects
        for non_delegatable in self.NON_DELEGATABLE[tool]:
            if llm_decision.get(non_delegatable) != user_intent.get(non_delegatable):
                return False  # LLM overstepped
        
        return True
```

**Example Flow:**
```
User: "Add 'Meeting at 3pm' to notes.txt"

LLM can decide:
- Format as "• Meeting at 3pm"
- Add timestamp: "• [2024-01-15] Meeting at 3pm"
- Add category: "#meeting • Meeting at 3pm"

LLM CANNOT decide:
```
