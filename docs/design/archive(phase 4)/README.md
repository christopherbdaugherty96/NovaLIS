# Legacy Phase 4 Archive

This folder preserves older Phase 4 and pre-Phase-4 design artifacts for traceability.

These files are:
- historical
- superseded
- not current roadmap authority

Do not use this folder as the starting point for Nova's active design direction.

Use instead:
- `docs/design/README.md`
- `docs/design/DESIGN_AUTHORITY.md`
- the relevant active phase folder under `docs/design/Phase */`

Important rule:

If anything in this folder conflicts with:
- `docs/current_runtime/CURRENT_RUNTIME_STATE.md`
- active phase document maps
- current corrected phase packets

the current runtime docs and active phase folders win.

This archive is retained because many of these files capture:
- historical constitutional framing
- older roadmap drafts
- earlier synthesis attempts
- useful source context for how Nova evolved

They are preserved for reference, not for live planning authority.
