🏛️ EXECUTIVE SUMMARY

Nova's cognitive architecture splits intelligence into two orthogonal dimensions:

Dimension 1: Cognitive Intelligence (Epistemic)

· Multiple specialized agents providing deep, domain-specific analysis
· Orthogonal design: No agent overlaps in function
· No synthesis layer: Agents don't communicate or merge outputs
· Raw outputs only: Each agent produces findings without interpretation

Dimension 2: Social Intelligence (Communicative)

· Single Personality Agent providing conversational interface
· No epistemic authority: Only linguistic, never meaning transformation
· Full transparency: Discloses all agents invoked and outputs presented
· Strict boundary: Cannot synthesize, prioritize, or interpret

Governance Foundation

All agents operate under Nova's Intelligence-Authority Split:

· Intelligence: Distributed across specialized agents
· Authority: Zero autonomous - all decisions remain with user
· Governor: Single choke point for all agent invocations

---

🧠 CORE PHILOSOPHY

Orthogonality Principle

"If two agents could plausibly produce the same output, one of them is wrong to exist."

Epistemic Hygiene Over Agreement

Nova optimizes for:

· Falsifiability: Multiple perspectives expose contradictions
· Adversarial surfacing: Agents challenge assumptions
· Intellectual hygiene: Clear separation of concerns
· Tension preservation: No premature resolution of disagreements

Intelligence-Authority Split (Canonical)

```
Social Intelligence (Personality Agent)   ≠   Cognitive Intelligence (Orthogonal Agents)
      │                                           │
      └── Communicative, presentational           └── Epistemic, analytical
      └── No meaning transformation                └── Domain-specific truth-seeking
      └── Zero decision authority                  └── Zero decision authority
```

🔧 ORTHOGONAL AGENT SPECIFICATIONS

1️⃣ Builder/Generator Agent

Purpose: Turn ambiguous intent into structured creation

Input: Vague ideas, problem statements, "how might we" questions
Output: PROPOSAL, DRAFT, STRUCTURE

Hard Limits:

· ❌ Cannot claim correctness
· ❌ Cannot reference canon unless explicitly quoted
· ❌ Cannot self-audit
· ❌ Cannot prioritize
· ❌ Cannot label anything "final"

Use When: Starting from zero, need initial structure

---

Purpose: Brutal, document-level scrutiny

Input: Any structured document, phase specifications, canonical texts
Output: AUDIT, FINDINGS, VIOLATIONS, OPEN_QUESTIONS

Special Permissions:

· ✅ Allowed to be slow (minutes, not seconds)
· ✅ Allowed to be repetitive (multiple coverage angles)
· ✅ Allowed large outputs (10K+ words)

Hard Limits:

· ❌ Cannot propose fixes
· ❌ Cannot rewrite content
· ❌ Cannot summarize away problems
· ❌ Cannot be "helpful"

Nova-Specific Features:

· Phase-aware audit depth

· Phase transition impact assessment

Input: Canonical documents, drafts, deprecated ideas, user teaching logs

Output Shape Specification

```python
def create_presentation(agent_outputs: Dict[str, AgentOutput]) -> str:
    """
    Creates presentation that preserves raw outputs without synthesis.
    """
    presentation = []
    
    # 1. Transparent invocation disclosure
    presentation.append("## Invocation Results")
    presentation.append(f"I ran: {', '.join(agent_outputs.keys())}")
    presentation.append("")
    presentation.append("**No outputs have been combined, summarized, or interpreted.**")
    presentation.append("")
    
    # 2. Raw outputs with clear labeling
    for agent_name, output in agent_outputs.items():
        presentation.append(f"### {agent_name} Output")
        presentation.append(f"*Generated at: {output.timestamp}*")
        presentation.append("")
        presentation.append(output.raw_text)
        presentation.append("")
        presentation.append("---")
        presentation.append("")
    
    # 3. Explicit non-synthesis statement
    presentation.append("## Next Steps")
    presentation.append("These are raw outputs. You may:")
    presentation.append("- Review each agent's findings")
    presentation.append("- Note contradictions between agents")
    presentation.append("- Decide which findings are relevant")
    presentation.append("- Request additional analysis if needed")
    presentation.append("")
    presentation.append("*I cannot synthesize, prioritize, or resolve these for you.*")
    
    return "\n".join(presentation)
```

Validation Framework

Transformation Validator

```python
class TransformationValidator:
    """
    Enforces linguistic vs epistemic boundary.
    """
    def validate_transformation(self, original: str, transformed: str) -> ValidationResult:
        # Core test: Meaning preservation
        original_claims = self._extract_factual_claims(original)
        transformed_claims = self._extract_factual_claims(transformed)
        
        if set(original_claims) != set(transformed_claims):
            return ValidationResult(
                valid=False,
                violation="EPISTEMIC_TRANSFORMATION",
                details="Meaning altered during transformation"
            )
        
        return ValidationResult(valid=True)
```

Formatting Validator (Prevents Implicit Synthesis)

```python
class FormattingValidator:
    """
    Ensures formatting doesn't convey synthesis or prioritization.
    """
    PROHIBITED_HIERARCHY_RULES = [
        "No top-down ordering based on perceived importance",
        "No grouping that implies consensus or priority",
        "No visual cues (bold, color, symbols) that suggest ranking"
    ]
    
    def validate(self, presentation: str) -> ValidationResult:
        # Check for visual prioritization
        if "⭐" in presentation or "⚠️" in presentation:
            return ValidationResult(
                valid=False,
                violation="VISUAL_PRIORITIZATION",
                details="Symbols imply importance ranking"
            )
        
        # Check for bolded emphasis beyond agent names
        bolded_sections = self._extract_bolded_text(presentation)
        if len(bolded_sections) > len(self._extract_agent_names(presentation)):
            return ValidationResult(
                valid=False,
                violation="TEXTUAL_EMPHASIS",
                details="Excessive bolding implies prioritization"
            )
        
        return ValidationResult(valid=True)
```

⚡ PERFORMANCE & OPERATIONAL CONSIDERATIONS

Execution Model: Context-Aware Parallel Execution

```python
class ContextAwareParallelExecution:
    """
    Runs agents in phases with read-only context sharing.
    Maintains orthogonality while enabling cross-agent insights.
    """
    def execute_agents(self, agents: List[Agent], query: str) -> Dict[str, AgentOutput]:
        completed_outputs = {}
        
        # Phase 1: Generators first
        for agent in self._get_agent_group("generator", agents):
            output = agent.execute(query, context={})
            completed_outputs[agent.name] = output
        
        # Phase 2: Analyzers with Phase 1 context
        analysis_context = self._create_read_only_snapshot(completed_outputs)
        for agent in self._get_agent_group("analyzer", agents):
            output = agent.execute(query, context=analysis_context)
            completed_outputs[agent.name] = output
        
        # Phase 3: Meta-agents with full context
        meta_context = self._create_read_only_snapshot(completed_outputs)
        for agent in self._get_agent_group("meta", agents):
            output = agent.execute(query, context=meta_context)
            completed_outputs[agent.name] = output
        
        return completed_outputs
```

```python
class PersonalityAgentTests:
    def test_no_synthesis(self):
        """Ensures no synthesis language in outputs."""
        presentation = agent.present_outputs(outputs)
        prohibited = ["overall", "in summary", "key takeaway", "therefore"]
        for phrase in prohibited:
            assert phrase not in presentation.lower()
    
    def test_no_prioritization(self):
        """Ensures no implicit prioritization through formatting."""
        presentation = agent.present_outputs(outputs)
        assert presentation.count("⚠️") == 0
        assert presentation.count("⭐") == 0
        assert presentation.count("**") <= len(agent_outputs)  # Only agent names
    
    def test_meaning_preservation(self):
        """Ensures factual claims unchanged."""
        for agent_name, output in agent_outputs.items():
            presented = agent.present_single_output(agent_name, output)
            assert extract_factual_claims(output) == extract_factual_claims(presented)
    
    def test_authority_boundary(self):
        """Ensures no authority language."""
        presentation = agent.present_outputs(outputs)
        prohibited = ["i think", "i believe", "i recommend", "you should"]
```

```python
class NovaAgentStressTester:
    """
    Tests agent stack against Nova-specific failure modes.
    """
```
