# Nova Future Naming Note: GhOVLIS
Date: 2026-03-21
Status: Future naming note only
Scope: Preserve a possible later naming direction without changing the current Nova name or current runtime references

## Note
A future naming idea to preserve is:

`GhOVLIS`

Expanded as:

`GOV LOCAL INTELLIGENCE SYSTEM`

with:
- `H` standing for `Home`

Plain-language interpretation:
- a governed local intelligence system
- with home as an explicit part of the long-term identity

## Important Boundary
This is only a future naming note.

It does not mean:
- Nova should be renamed now
- current docs/runtime should be updated to that name now
- the existing Nova identity should be replaced in the current product baseline

This note only preserves the idea for later branding or naming review near the end of the roadmap, if that still feels right at that time.
