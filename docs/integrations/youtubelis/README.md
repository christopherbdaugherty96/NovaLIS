# YouTubeLIS Integration Concept

YouTubeLIS is a proposed governed content-production workflow for NovaLIS.

It is not a completed NovaLIS capability yet. This folder preserves the concept, operating model, and future integration direction so the idea can be developed without overstating current runtime support.

## Purpose

YouTubeLIS is designed to help produce high-quality videos about important AI and technology developments:

- AI breakthroughs
- The AI race
- Robotics
- Automation
- Job loss and job transformation
- New AI-enabled jobs
- Real-world impact of emerging tools

The intended channel style is:

> Real footage for proof + animation for explanation + narration for story.

## NovaLIS Fit

YouTubeLIS fits NovaLIS because it requires more than generation. It needs governed orchestration:

- Idea selection
- Research
- Claim checking
- Script drafting
- Visual planning
- Voice generation
- Editing workflow support
- Publishing review
- Performance feedback

NovaLIS should remain the governed coordinator, not an uncontrolled publisher.

## Non-Goal

YouTubeLIS is not intended to be a spam generator, autonomous upload farm, or unsupervised content bot.

## Governance Principle

> Maximum useful automation, not uncontrolled automation.

Any desktop, browser, OpenClaw, ElevenLabs, editing, upload, or publishing action should be task-scoped, explicitly approved when risk exists, logged, and stopped on completion.

## Current Status

Planning/integration concept only.

Do not describe this as shipped runtime functionality until code, tests, UI, and real workflow execution exist.
