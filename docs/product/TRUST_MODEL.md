# Trust Model

Last reviewed: 2026-04-28

Nova is built on a simple principle:

**Intelligence is not authority.**

A model may interpret, summarize, search, draft, or recommend. Real actions should pass through explicit controls.

## Plain-English Version

Nova aims to make these things clear:
- what was requested
- whether a real action is involved
- what capability would act
- whether the action is allowed, blocked, or needs approval
- what result occurred

## Why This Matters

Many systems optimize for invisible automation. Nova prioritizes visible control and reviewable behavior.
