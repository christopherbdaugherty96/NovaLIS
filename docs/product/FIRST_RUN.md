# First Run

Use this page after installing Nova for the first time.

## Step 1: Launch
```bash
nova-start
```
Open: `http://localhost:8000`

## Step 2: Try These 5 Commands
1. What works today?
2. Explain what you can do
3. Weather in Belleville
4. Summarize today's news
5. Draft an email to test@example.com

## Step 3: What To Check
- Does chat respond?
- Does the dashboard load?
- Are responses reasonably fast?
- Do simple actions stay clear and bounded?

## Step 4: If Something Fails
- Check terminal logs
- Confirm Ollama is running
- Confirm models are installed
- Review Known Limitations
- Review Runtime Truth docs
