# Known Limitations

This page exists to set honest expectations about Nova today.

## Product State
- Nova is still an active build, not a finished mainstream release.
- Some flows are polished while others are evolving.
- Setup still expects a technically comfortable user.

## Environment Dependence
- Local models must be installed and configured.
- Performance depends on your hardware.
- Some local actions vary by operating system.
- Optional connectors require additional setup.

## Current Gaps
- Trust UI is improving but not complete for every flow.
- One-click installer is not finished.
- Some advanced workflows remain experimental.
- Broad autonomous execution is intentionally limited.

## Best Fit Today
Nova is best for builders, early adopters, and users who value control more than maximum convenience.

## Why This Matters
Honest limits build trust better than inflated claims.
