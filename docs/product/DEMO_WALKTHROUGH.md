# Nova Demo Walkthrough

Use this guide to quickly demonstrate what Nova is today.

---

## Goal
Show useful AI assistance with visible boundaries and local-first trust.

---

## Demo Flow (5 Minutes)

### 1. Start Nova
Run:
```text
nova-start
```
Open the dashboard.

### 2. Show Daily Value
Ask:
```text
daily brief
```
Show summaries and structured output.

### 3. Show Research Help
Ask:
```text
research local AI tools for small businesses
```
Show reasoning and usable findings.

### 4. Show Safe Local Actions
Ask:
```text
open downloads
mute
```
Explain that actions are bounded and visible.

### 5. Show Draft-Only Help
Ask:
```text
draft an email to test@example.com about a project update
```
Explain that Nova helps draft but does not send autonomously.

### 6. Show Memory / Continuity
Ask:
```text
remember this: client prefers morning calls
memory overview
```
Show continuity value.

---

## Core Message
Nova is useful because it helps with real work while keeping the user in control.
