# Demo Script

Last reviewed: 2026-04-28

## 60 Second Nova Demo

1. Open Nova dashboard.
2. Ask: Summarize today's news.
3. Ask: Draft an email to John about tomorrow.
4. Show governance / trust review surface.
5. Show result or receipt.
6. Close with: Intelligence is not authority.

## Goal

Help a first-time visitor understand Nova in under one minute.
