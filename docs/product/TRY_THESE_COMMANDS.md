# Try These Commands

Use these prompts to quickly understand Nova's current value.

## Everyday Utility
- What works today?
- What's the weather in Belleville?
- Summarize today's news
- Explain what Nova can do

## Trust / Governance
- Why was that action blocked?
- Show recent trust receipts
- Explain how actions are controlled
- What capabilities are active?

## Productivity
- Draft an email to John about tomorrow
- Summarize this topic for me
- Help me plan my day
- Turn this into a checklist

## Exploration
- Explain what is on my screen
- Search the web and cite sources
- Compare these two ideas
- Give me a second opinion on this plan

## Reminder
Some capabilities depend on local setup, credentials, or operating system support.
Use runtime truth docs for exact live capability status.
