# User Ready Status

This page explains what Nova feels ready for today, what is advanced, and what is still future work.

---

## Ready Now
- Core assistant conversations
- Daily briefs, summaries, and research help
- Dashboard and runtime surfaces
- Memory and continuity features
- Some local device actions (depends on setup)
- Documentation-driven local usage

## Advanced / Experimental
- Screen capture and screen analysis flows
- Connector-based workflows
- OpenClaw worker surfaces
- Some higher-risk or mutation workflows
- Deeper operator automation paths

## Requires Extra Setup
- Ollama / local models
- Optional API keys or connectors
- Voice models and local speech tools
- OS-specific local controls
- Calendar `.ics` configuration

## Not Yet Finished
- Fully polished trust UI for every action
- One-click installer
- Guided first-run onboarding
- Consumer-grade simplicity
- Broad release readiness

## Honest Guidance
If you are comfortable with local tools and evolving software, Nova may already be useful.
If you expect a finished mainstream product, Nova is still in progress.
