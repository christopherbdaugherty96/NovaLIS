# docs/todo — Task Folder

This folder holds actionable task lists only. No wishlists, no design docs, no strategy.

| File | Purpose |
|------|---------|
| [ACTIVE_TODO.md](ACTIVE_TODO.md) | Current sprint tasks ordered by ROI |
| [TECH_DEBT.md](TECH_DEBT.md) | Technical debt and reliability items |
| [DOC_CLEANUP.md](DOC_CLEANUP.md) | Documentation maintenance tasks |

---

## What belongs here

- concrete next actions with a clear done state
- blockers and their specific unblock conditions
- debt items with file-level scope

## What does NOT belong here

- product vision → `docs/future/`
- strategic direction → `docs/future/`
- quality standards → `docs/future/`
- future feature design → `docs/future/`
- active sprint detail → `4-15-26 NEW ROADMAP/Now.md`
- strategy and tier roadmap → `4-15-26 NEW ROADMAP/MasterRoadMap.md`
- deferred work → `4-15-26 NEW ROADMAP/BackLog.md`
