# рџ§  Backend Components

### **brain_server.py**
Handles:
- WebSocket bidirectional communication  
- Skill routing  
- AI responses  
- Memory storage  
- Widget updates (future)  

### **nova_config.py**
Central configuration:
- Model settings  
- Port settings  
- Skills enable/disable  
- ORB configuration  
- System prompt  

### **nova_memory.py**
Implements:
- SQLite conversation history  
- Preference storage  
- Future: long-term memory  

### **skill_registry.py**
Auto-discovers skills inside `/skills` using dynamic imports.

### **nova_tools.py**
Decides whether a user message:
- Should trigger a skill  
- Or be sent to the AI model
