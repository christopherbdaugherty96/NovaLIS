# Status Notes

This folder contains human-maintained work-continuity notes.

These notes are useful for handoffs, recent-session recovery, and current-priority tracking.

They are not generated runtime truth.

Use generated runtime docs for exact live capability/runtime state:

- [`../current_runtime/CURRENT_RUNTIME_STATE.md`](../current_runtime/CURRENT_RUNTIME_STATE.md)
- [`../current_runtime/RUNTIME_CAPABILITY_REFERENCE.md`](../current_runtime/RUNTIME_CAPABILITY_REFERENCE.md)

Use this folder to understand what recent work was completed, what remains local or in-progress, and what should not be overstated.
