# Nova Creator-Led Shopify POD Model

Date: 2026-05-11
Status: planning only / non-authorizing / future direction

## Purpose

Define the intended operating model for a creator-led Shopify + print-on-demand business operated with Nova as a governed business-intelligence layer.

This is not runtime authority.

This document does not:
- add capabilities
- add external writes
- add autonomous workflows
- add browser/computer-use
- expand OpenClaw authority
- approve financial automation
- approve autonomous marketing
- approve autonomous publishing

The governing rule remains:

```text
Intelligence is not authority.
```

---

## Operating Model

```text
Christopher = creator / art direction / approvals
Nova = reporting / analytics / drafting / recommendations
Shopify = commerce system
Printful = fulfillment + manufacturing
```

The intended business structure is:

```text
creator-led brand
+ governed business intelligence
+ third-party fulfillment
```

not autonomous e-commerce operation.

---

## Recommended Initial Stack

### Commerce

- Shopify

### Fulfillment

- Printful

### Optional later expansion

- Gelato
- Prodigi

for premium poster and framed-print workflows.

---

## Nova Role

Nova should operate as a read-only business-intelligence and planning layer.

### Allowed responsibilities

- sales summaries
- product-performance analysis
- marketing analytics
- finance summaries
- trend detection
- campaign drafting
- caption drafting
- product-description drafting
- launch checklists
- customer/order summaries
- inventory visibility
- design backlog organization
- KPI reporting
- strategic recommendations

### Explicitly not approved

- autonomous product publishing
- autonomous price changes
- autonomous ad spend
- autonomous email sends
- autonomous customer messaging
- autonomous refunds
- autonomous Shopify setting changes
- autonomous social posting
- autonomous financial transactions
- autonomous inventory ordering

Nova prepares recommendations.
Christopher approves actions.

## Intended Daily Workflow

```text
Christopher creates designs
        ↓
Products published through Shopify + Printful
        ↓
Nova analyzes store metrics
        ↓
Nova prepares summaries and recommendations
        ↓
Christopher approves next actions
```
