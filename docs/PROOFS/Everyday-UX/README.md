## Purpose

This proof folder tracks whether Nova is usable as a daily tool for a normal person.
It records conversational friction, UI friction, and live workflow results — not capability
certification. Capability certification lives in `docs/capability_verification/`.

---

## Scope constraints

This folder does not contain evidence for:

- new capability certification
- OpenClaw workflow proofs
- browser/computer-use proofs
- external write proofs

Those belong in `docs/PROOFS/Phase-*/` or `docs/capability_verification/`.
