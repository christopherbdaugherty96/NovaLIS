# Brain Dry-Run Examples

Date: 2026-04-29

These examples apply the Brain architecture as a read-only planning scaffold.

They do not execute actions, alter Governor routing, send email, write Shopify data, or start OpenClaw automation.

Use them to compare expected Brain reasoning against current Nova runtime behavior.

## Files

- `EXAMPLES.md` - dry-run examples for local conversation, web search, email draft, Shopify, OpenClaw, and personal browser/account tasks
- `PROOF_INDEX.md` - index of the dry-run proof artifacts
