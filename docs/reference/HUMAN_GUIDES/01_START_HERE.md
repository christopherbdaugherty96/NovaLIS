## What Nova Is
Nova is best understood as a personal intelligence workspace that lives on your computer.

It is designed to help with:
- research
- explanation
- computer guidance
- ongoing projects
- continuity within the current workspace session
- explicit governed memory
- durable context across sessions when you choose to save it

Nova is not just a chatbot.
It is also not meant to be an unconstrained AI agent.

The idea behind Nova is that it should be able to think, explain, and organize work without quietly taking over authority.

## What Makes Nova Different
Most assistants are built around a simple pattern:
- user asks something
- model answers
- sometimes model acts directly

Nova is built around a different pattern:
- user asks
- Nova interprets the request
- Nova either analyzes or uses a governed capability
- any real action passes through a control path
- actions are bounded and logged

This gives Nova a different feel.
It is meant to be:
- thoughtful
- inspectable
- stable
- explicit
- easier to trust

## What Nova Feels Like When It Works Well
In everyday use, Nova is meant to feel like:
- a calm intelligence layer over your computer
- a system that can explain what you are seeing
- a system that can help you continue real work
- a system that can preserve important decisions without hidden persistence

Examples of that feeling:
- `Nova, what is this page?`
- `Nova, summarize story 2.`
- `Nova, continue my deployment issue.`
- `Nova, what is my most blocked project?`
- `Nova, memory save decision for deployment issue: use PYTHONPATH inspection first.`

## What Nova Is Not
Nova is not designed to be:
- a hidden background monitor
- a self-starting autonomous agent
- a silent persistence system
- a black-box action engine

If Nova captures the screen, reads a page, opens something, or saves memory, that should happen because the user asked.

## The Short Version
If you want the simplest possible description, this is it:

Nova is a governed intelligence system that helps you understand information, continue work, and preserve important context without giving up user control.
