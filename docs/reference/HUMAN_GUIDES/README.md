# Nova Human Guides
Updated: 2026-03-30
Status: Explanatory guide set

## What These Documents Are
This folder is the plain-language explanation of Nova.

It is for people who want to understand:
- what Nova is
- what Nova can do
- how it works at a high level
- what is live today
- what is still planned
- how the continuity and memory system fit together
- why Nova is designed to stay safe

These guides are intentionally written in normal human language.
They are not code documents, not phase locks, and not constitutional law.

## What These Documents Are Not
These guides are not the source of runtime truth.
If there is ever a conflict between these guides and the live runtime, the runtime truth docs win.

Runtime truth lives in:
- `docs/current_runtime/CURRENT_RUNTIME_STATE.md`
- `docs/current_runtime/RUNTIME_CAPABILITY_REFERENCE.md`
- `docs/current_runtime/RUNTIME_FINGERPRINT.md`

## Quick Summary
Nova is a governed personal intelligence system.

That means it is built to:
- think deeply
- explain clearly
- help with ongoing work
- stay under explicit user control

Nova is not supposed to quietly turn into an autonomous agent.
Its core design rule is simple:

`Intelligence may expand. Authority may not expand without explicit unlock.`
