## Current Accepted State

- Nova is a governance-first local AI/runtime platform.
- Intelligence is not authority.
- Memory does not grant permission.
- Obsidian can help Nova understand, but Obsidian cannot authorize Nova to act.
- Knowledge is context, not permission.
- Notes are knowledge source, not execution proof.
- Goal Cards remain display-only and do not execute tasks.
- Second Brain Slice 1 priority lock is accepted and limited to schema,
  frontmatter parser, wikilink extraction, vault health/lint, no-mutation tests,
  and non-authorizing tests only.
- No vector DB, MCP, dashboard graph, memory promotion, proposal writes,
  execution integration, scheduler, OpenClaw integration, or capability
  expansion is authorized by the current priority.
