# Nova Governance Boundaries

Status: Template
Date: 2026-06-08
Source: AI Ecosystem Operating Model
Authority Tier: Tier 3 draft/template
Review Required: Yes

Core rules:

- Intelligence is not authority.
- Memory does not grant permission.
- Implementation does not grant execution authority.
- Human approval remains the highest business authority.
- Nova governance controls execution authority.
- Receipts and logs prove runtime behavior.
