    Introduction

    Welcome to Nova

    Offline-first

    Nova is a personal intelligence system designed to help you think, research, and interact with your computer while keeping you in full control.

    Local Mode: Nova stays local-first, private, and cost-free by default.

    What Nova Does

    Research, summarize, explain, and help you work through decisions without hiding what it used.

    • Research and summarize information
    • Explain documents, code, or on-screen content
    • Help you think through decisions
    • Control parts of your system when explicitly allowed
    • Provide structured insights and reports
    Core Principles

    Nova is powerful because it stays bounded.

    You Are In Control
    Every action must be explicitly requested or approved.
    Transparency First
    You can see what Nova is doing, what data is being used, and what tools are being called.
    Bounded Intelligence
    Nova can reason and assist, but it cannot act autonomously or run hidden background work.
    How Nova Works

    Every request follows the same simple flow.

    You -> Nova -> Validation -> Execution -> Result

    Before anything happens, Nova checks permissions, verifies scope, and enforces limits. If something is not allowed, it is blocked or requires approval.

    Setup Options

    You can stay fully local or choose a future cloud path later in Settings.

    Local Mode
    Uses local AI models, needs no API key, and is the safest current everyday setup.
    Bring Your Own API Key
    Lets you control provider, usage, and billing once you explicitly connect one.
    Managed Cloud Access
    Will be the easiest future setup path, but should still stay visible and explicit.
    Security, Privacy, and First Steps

    Nova should feel strong without ever becoming sneaky.

    • No hidden actions, no background monitoring, and no continuous listening
    • Paid APIs should never be used without your knowledge
    • Sensitive actions may be blocked or require confirmation and are always logged
    • Start with Home, then Workspace, then Trust if you want to understand what happened
    • Try commands like "Summarize today's news", "Explain this file", or "Help me think through this idea"

    Nova is built on a simple idea: intelligence should be powerful, but control should never be lost.
